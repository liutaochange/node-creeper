### 一个基于node的爬虫项目练习

#### 技术栈

1. express     启动服务,监听路由
2. request     页面数据下载
3. cheerio     页面数据解析

#### 安装教程

``` bash
# clone project
git clone https://github.com/liutaochange/node-creeper.git

# install dependencies
npm install

# serve with hot reload at localhost:3000
node app.js 或者 npm run start

```
